## 新年度の準備手順

### ステップ1: データの準備

#### 1-1. Excelファイルの作成

1. `data/students.xlsx` を作成または更新
2. 必須項目を入力：
   - No（学生番号）
   - 所属学年（メデ4）
   - 学籍番号（7桁の数字）
   - 氏名（日本語）
   - 氏名英字（ローマ字）
   - 題目（研究題目）
   - 画像パス（画像ファイル名）
   - レポートパス（Wordファイル名）
   - プレゼンパス（PPTXファイル名）

**詳細**: [docs/EXCEL項目一覧.md](EXCEL項目一覧.md) を参照

#### 1-2. ファイルの配置

以下のフォルダ構造に従ってファイルを配置します：

```
assets/
├── images/
│   └── [学籍番号]/          # 例: 1522008/
│       └── image1.png       # 画像ファイル
├── reports/
│   └── pdf/
│       └── [学籍番号].pdf   # 例: 1522008.pdf
├── presentations/
│   └── master/
│       └── pdf/
│           └── [学籍番号].pdf  # 例: 1522008.pdf
└── hero/
    └── videos/
        └── [学籍番号].mp4   # オプション: Hero動画
```

**注意**: ファイル名は学籍番号で統一してください。

#### 1-3. PDFファイルの変換

レポートやプレゼンテーション資料をPDFに変換します：

```bash
# レポート（Word → PDF）
python3 scripts/convert/convert_reports.py

# プレゼンテーション（PPTX → PDF）
python3 scripts/convert/process_final_presentations.py
```

**詳細**: [scripts/FINAL_PRESENTATION_GUIDE.md](../scripts/FINAL_PRESENTATION_GUIDE.md) を参照

#### 1-4. JSONファイルの生成

ExcelファイルからJSONファイルを生成します：

```bash
python3 scripts/excel/convert_excel_to_json.py
```

これにより `data/students.json` が生成されます。

### ステップ2: 年度設定の更新

#### 2-1. 新年度を追加

年度管理スクリプトを使用して新年度を追加します：

```bash
# 例: 2026年度、発表会日: 2027-01-27
python3 scripts/manage_year.py add-year --year 2026 --presentation-date 2027-01-27
```

**パラメータ説明**:
- `--year`: 年度（例: 2026）
- `--presentation-date`: 発表会日（YYYY-MM-DD形式）
- `--hide-after-date`: 個人情報非表示開始日（省略時は発表会日の翌日）

**実行結果**:
- `data/config.json` が自動更新されます
- 現在年度が新年度に更新されます

#### 2-2. 設定の確認

設定が正しく追加されたか確認します：

```bash
python3 scripts/manage_year.py show-config
```

**確認ポイント**:
- 現在年度が正しく設定されているか
- 発表会日が正しいか
- 非表示開始日が発表会日の翌日になっているか

## 発表会後の対応手順

### ステップ1: 個人情報を非表示にする

発表会の翌日に、個人情報を自動的に非表示にする設定を有効化します：

```bash
# 例: 2025年度の個人情報を非表示にする
python3 scripts/manage_year.py hide-info --year 2025
```

**実行結果**:
- `data/config.json` の `hidePersonalInfo` が `true` に設定されます
- ウェブサイト上で氏名と学籍番号が非表示になります

**注意**: この設定は発表会日の翌日に自動的に適用されるように設計されていますが、手動で実行することも可能です。

---

### ステップ2: プライバシーアラートの削除（オプション）

`index.html` のプライバシーアラートがコメントアウトされている場合、削除またはそのままにしておきます。

現在はコメントアウトされているため、そのままで問題ありません。

---

### ステップ3: 動作確認

個人情報が正しく非表示になっているか確認します：

- [ ] 学生一覧で氏名が非表示になっている
- [ ] 学籍番号が非表示になっている
- [ ] 詳細ページでも氏名が非表示になっている
- [ ] 研究題目や画像は表示されている

## 過去年度のアーカイブ手順

年度が終了したら、過去年度のデータをアーカイブに移動します。

### ステップ1: データをアーカイブに移動

```bash
# 例: 2025年度のデータをアーカイブに移動
python3 scripts/manage_year.py archive --year 2025
```

**実行結果**:
- `data/students.json` → `data/archive/students_2025.json` に移動
- `data/config.json` の `dataFile` が更新されます

---

### ステップ2: アセットファイルの整理（オプション）

必要に応じて、過去年度のアセットファイルを別フォルダに移動します：

```bash
# 例: 2025年度の画像をアーカイブフォルダに移動
mkdir -p assets/archive/2025/images
mv assets/images/152* assets/archive/2025/images/

# PDFファイルも同様に
mkdir -p assets/archive/2025/reports/pdf
mkdir -p assets/archive/2025/presentations/master/pdf
mv assets/reports/pdf/152*.pdf assets/archive/2025/reports/pdf/
mv assets/presentations/master/pdf/152*.pdf assets/archive/2025/presentations/master/pdf/
```

**注意**: アセットファイルの移動は必須ではありません。ディスク容量に余裕がある場合は、そのまま残しておいても問題ありません。

## トラブルシューティング

### 年度設定が反映されない

**原因**: `data/config.json` が正しく更新されていない

**解決方法**:
```bash
# 設定を確認
python3 scripts/manage_year.py show-config

# 必要に応じて再設定
python3 scripts/manage_year.py add-year --year [年度] --presentation-date [日付]
```

---

### 個人情報が非表示にならない

**原因**: `hidePersonalInfo` が `false` のまま

**解決方法**:
```bash
# 手動で非表示設定を有効化
python3 scripts/manage_year.py hide-info --year [年度]

# 設定を確認
python3 scripts/manage_year.py show-config
```

---

### JSONファイルが読み込めない

**原因**: `data/students.json` が存在しない、または形式が不正

**解決方法**:
```bash
# ExcelからJSONを再生成
python3 scripts/excel/convert_excel_to_json.py

# JSONファイルの形式を確認（エディタで開いて確認）
```

---

### 画像やPDFが表示されない

**原因**: ファイルパスが間違っている、またはファイルが存在しない

**解決方法**:
1. ファイルが正しいフォルダに配置されているか確認
2. JSONファイル内のパスが正しいか確認（ファイル名のみでOK）
3. ファイル名に日本語や特殊文字が含まれていないか確認

## 補足: 設定ファイルの構造

`data/config.json` の構造：

```json
{
  "currentYear": 2026,
  "years": [
    {
      "year": 2025,
      "presentationDate": "2026-01-27",
      "hidePersonalInfoAfter": "2026-01-28",
      "hidePersonalInfo": true,
      "dataFile": "archive/students_2025.json"
    },
    {
      "year": 2026,
      "presentationDate": "2027-01-27",
      "hidePersonalInfoAfter": "2027-01-28",
      "hidePersonalInfo": false,
      "dataFile": "students.json"
    }
  ]
}
```

**フィールド説明**:
- `currentYear`: 現在の年度
- `years`: 年度別設定の配列
  - `year`: 年度
  - `presentationDate`: 発表会日（YYYY-MM-DD形式）
  - `hidePersonalInfoAfter`: 個人情報非表示開始日（YYYY-MM-DD形式）
  - `hidePersonalInfo`: 個人情報を非表示にするか（true/false）
  - `dataFile`: データファイルのパス（現在年度は `students.json`、過去年度は `archive/students_[年度].json`）
