# Excelファイル項目一覧

## 必須項目

以下の列をExcelファイルの1行目（ヘッダー行）に配置してください。

| 列名 | 説明 | 例 | 備考 |
|------|------|-----|------|
| **No** | 学生番号（連番） | 1, 2, 3... | 必須。各学生を識別するための番号 |
| **所属学年** | 所属学年 | メデ4 | 必須 |
| **学籍番号** | 学生の学籍番号 | 1522008 | 必須 |
| **氏名** | 学生の氏名（日本語） | 岩舩　愛 | 必須 |
| **氏名英字** | 学生の氏名（ローマ字） | AI IWAFUNE | 必須 |

## 推奨項目

| 列名 | 説明 | 例 | 備考 |
|------|------|-----|------|
| **題目** または **研究題目** | 卒業研究の題目 | 「AIを活用した画像認識システムの開発」 | どちらかの列名を使用可能 |
| **画像パス** または **画像** | 画像ファイル名（複数可） | `image1.jpg` または `image1.jpg,image2.jpg` | カンマ区切りで複数指定可能 |
| **レポートパス** または **レポート** | Wordレポートファイル名（複数可） | `report.docx` または `report1.docx,report2.docx` | カンマ区切りで複数指定可能 |
| **プレゼンパス** または **プレゼン** または **プレゼンテーション** | PPTXプレゼン資料ファイル名（複数可） | `presentation.pptx` または `pres1.pptx,pres2.pptx` | カンマ区切りで複数指定可能 |

## ファイルパスの指定方法

### 画像ファイル
- **ファイル名のみ指定**: `image1.jpg` → `assets/images/image1.jpg` として読み込まれます
- **複数ファイル**: `image1.jpg,image2.jpg,image3.jpg` （カンマ区切り）
- **フルURL**: `https://example.com/image.jpg` （外部URLも使用可能）

### レポートファイル
- **ファイル名のみ指定**: `report.docx` → `assets/reports/report.docx` として読み込まれます
- **複数ファイル**: `report1.docx,report2.docx` （カンマ区切り）

### プレゼンテーション資料
- **ファイル名のみ指定**: `presentation.pptx` → `assets/presentations/presentation.pptx` として読み込まれます
- **複数ファイル**: `pres1.pptx,pres2.pptx` （カンマ区切り）

## 注意事項

1. **列名の大文字小文字**: 列名は正確に一致させる必要があります（例: `No` は `no` や `NO` では認識されません）

2. **ファイル名の規則**:
   - ファイル名に日本語や特殊文字を含めないことを推奨します
   - スペースは避け、アンダースコア（`_`）やハイフン（`-`）を使用してください
   - 例: `学生1の画像.jpg` ❌ → `student1_image.jpg` ✅

3. **ファイルの配置場所**:
   - 画像: `assets/images/` フォルダ
   - レポート: `assets/reports/` フォルダ
   - プレゼン資料: `assets/presentations/` フォルダ

4. **空欄の扱い**:
   - 画像、レポート、プレゼン資料が無い場合は、セルを空欄にしておけば非表示になります

5. **複数ファイルの指定**:
   - カンマ区切りで複数のファイルを指定できます
   - 画像の場合、最初の画像が一覧ページのサムネイルとして使用されます

## 推奨されるExcelファイル名

- `students.xlsx` （`data/` フォルダに配置）
- または `data.xlsx`
- または `卒業研究データ.xlsx`

## テンプレートのダウンロード

以下の構造でExcelファイルを作成してください：

1. 1行目にヘッダー（列名）を入力
2. 2行目以降に学生データを入力
3. ファイルを `data/students.xlsx` として保存

## トラブルシューティング

### Excelファイルが読み込めない場合
- ファイルが `data/` フォルダに正しく配置されているか確認
- ファイル名が `students.xlsx` になっているか確認
- 列名が正確に一致しているか確認（スペースや全角半角に注意）

### 画像が表示されない場合
- ファイルが `assets/images/` フォルダに存在するか確認
- Excel内のファイル名と実際のファイル名が一致しているか確認
- ファイル名に日本語や特殊文字が含まれていないか確認
